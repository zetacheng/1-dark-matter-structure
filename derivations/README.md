# Derivations

Every derivation note must be committed before the production code that implements it and must contain:

- the scientific question;
- assumptions;
- applicable locked conventions;
- the analytic derivation;
- expected limiting cases;
- dimensional analysis;
- known failure modes;
- pre-registered verdicts, including the kill criterion;
- the exact scripts that implement it.

Derivation notes are historical scientific records. Corrections must be documented rather than silently rewriting the record.
